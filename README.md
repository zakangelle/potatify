# potatify

An image processing app to turn your hi-res images to potato quality.

[View site](http://potatify.herokuapp.com/)

## Requirements

+ NodeJS
+ ImageMagick
+ GraphicsMagick

## Setup

Install dependencies:

```sh
$ npm install
```

## Usage

Run the app at [http://localhost:8080](http://localhost:8080):

```sh
$ npm start
```
